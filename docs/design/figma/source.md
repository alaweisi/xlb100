## Permission / Limitation Record

- The Figma MCP source was readable.
- Root metadata is large and direct metadata output may be truncated by the model context window.
- A read-only Figma Plugin API scan was used to avoid guessing frame counts.
- Formal Figma components, published local styles, and local variable collections were not present in the inspected file.
- Short-lived Figma screenshot asset URLs were used only for download and are intentionally not stored in this repository.

## Security Boundary

Do not commit Figma tokens, MCP credentials, personal access keys, private screenshot URLs, or local MCP configuration. This repository stores only source identifiers, local PNG snapshots, and derived design documentation.
