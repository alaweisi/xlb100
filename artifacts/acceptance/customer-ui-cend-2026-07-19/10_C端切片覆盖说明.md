# C 端切片覆盖说明

本轮通过顾客端路由壳、共享 UI 组件、页面模板、状态组件和核心流程页面，将统一视觉基线覆盖到 C 端 10 个页面载体、62 个业务切片。

| 载体 | 用户任务 | 本轮覆盖重点 |
| --- | --- | --- |
| C-00 登录门禁 | 安全进入顾客端 | 加载门禁、中文反馈、品牌视觉 |
| C-01 首页 | 查找上门服务 | 城市与搜索、热门服务、加载/错误/空状态 |
| C-02 服务列表 | 筛选并选择服务 | 搜索、分类标签、服务卡片、无结果与重试 |
| C-03 服务预约 | 服务、地址、时间、确认、提交 | 四步进度、表单校验、固定 CTA、服务端成功 |
| C-04 我的订单 | 查看订单并执行后续动作 | 统一导航、订单内容层级、状态与操作区域 |
| C-05 售后服务 | 发起或查看售后 | 顾客端视觉基线、状态与恢复入口 |
| C-06 客服中心 | 创建并跟进客服请求 | 表单、列表、错误与后端确认反馈 |
| C-07 消息中心 | 阅读和处理通知 | 未读、归档、恢复、分页状态与中文文案 |
| C-08 优惠券 | 查看可用与历史权益 | 标签、卡片、空/错误状态与选择入口 |
| C-09 我的 | 管理资料与服务地址 | 资料加载、编辑、校验、保存和删除确认 |

## 共享覆盖能力

- 五项底部主导航和中心突出操作。
- 顶部标题、返回行为与安全区域。
- 暖米色、深绿色、橙色主操作及轻量玻璃层级。
- 卡片、表单、按钮、标签、搜索、Tabs、状态面板与 Action Dock。
- `44 × 44px` 最小触控目标、键盘焦点、减少动态和高对比度回退。
- 加载、空、错误、就绪、校验、提交中和服务端成功状态。

## 本包的渲染证据范围

浏览器重点验收覆盖核心预约流程以及共享结构风险最高的首页、服务列表和四类关键状态。其余切片通过共享组件覆盖和相关组件测试进行回归验证。
